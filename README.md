# Executar
1. clonar repositorio
2. executar
...

yarn install
..,

3 ter Nest CLI instalado
...

npm i -g @nestjs/cli
...

4. Levantar a base de dados
...

docker compose up -d

5. Reconstruir base de dados
...
http://localhost:3000/api/v2/seed
..

## Stack usado
* MongoDB
* Nest
